> WHAT'S IN THE BOX
Six core modules. Each one built to do one thing exceptionally well. Compose them together or use standalone.
api-engine.rs
API Engine

Define endpoints in minutes. Auto-generated docs, rate limiting, versioning, and type-safe SDKs out of the box.

webhooks.yaml
Real-Time Webhooks

Send and receive webhooks with automatic retries, signature verification, and real-time delivery logs.

query-builder.sql
Query Builder

Type-safe database queries without the ORM bloat. Raw SQL when you want it, abstractions when you don't.

auth.config.ts
Auth & Permissions

JWT, API keys, OAuth, RBAC — all built in. Define who can do what with a simple config file.

monitor.sh
Monitoring & Alerts

Real-time dashboards, error tracking, latency monitoring. Get paged on Slack or PagerDuty when things break.

teams.json
Team Workspaces

Shared environments, branch previews, audit logs, and granular permissions. Built for teams of 2 or 200.

import { Pipekit } from '@pipekit/core'

const app = new Pipekit()

// Define an endpoint in 4 lines
app.route('GET /users/:id', async (ctx) => {
  const user = await ctx.db.find('users', ctx.params.id)
  return ctx.json(user)
})

// Deploy with one command
app.listen({ port: 3000 })

console.log('→ Server running on :3000')

> FAQ
Pipekit is a backend toolkit for building APIs, webhooks, and data pipelines. Think of it as the missing framework between "raw Express" and "full-blown AWS." You get production-grade infrastructure with a developer experience that doesn't make you want to quit.
TypeScript and JavaScript are first-class citizens with full SDK support. Python and Go SDKs are in beta. Our REST API works with any language — if it can make HTTP requests, it works with Pipekit.
Enterprise plans include self-hosted and on-prem deployment options. For most teams, our managed cloud is the fastest path to production — we handle the infra so you can focus on your product.
We have migration guides for Express, Fastify, NestJS, and Django. Most teams migrate in under a week. Our CLI includes a `pipekit migrate` command that auto-detects your existing routes and generates scaffolding.
SOC 2 Type II certified. All data encrypted at rest and in transit. Automatic dependency scanning, RBAC by default, and audit logs on every action. We take security seriously because our users' users depend on it.
Free tier gets community Discord support. Pro gets priority email with 4-hour response times. Enterprise gets a dedicated support engineer and private Slack channel. We also have comprehensive docs, tutorials, and example repos.
